# About
Simple single-core hash cracker

# Usage
```
>>> python3 cracker.py --hash hash digest --algorithm hashing algorithm --wordlist wordlist
```

Example:
```
>>> python3 cracker.py --hash 7b5f3da4bc95956f8a7f8d6abf38afb6 --algorithm md5 --wordlist wordlist.txt


[!] Hash broken [!]
Hash: 7b5f3da4bc95956f8a7f8d6abf38afb6
Text: pee
Iteration: 6


```
